# Methodology

## Dataset: Interactive mini-papers: Historical suicide statistics in Spain before and after COVID-19. DOI: 10.17632/cwm4r6ywsc.1

**Author:** Juan Moisés de la Serna Tuya  
**ORCID:** https://orcid.org/0000-0002-8401-8018  

---

## 1. Data Collection

### Sources
Data was compiled from publicly available secondary sources including:
- International databases and organizations
- Peer-reviewed publications
- Official government statistics
- Open access repositories

### Process
1. Systematic search of relevant databases
2. Standardization of variable names and formats
3. Quality control and validation
4. Documentation of all sources

## 2. Data Processing

- **Format:** Structured tabular data (CSV/HTML)
- **Encoding:** UTF-8
- **Missing Values:** Coded as NA or empty cells
- **Units:** Documented in data dictionary

## 3. Quality Control

- Cross-validation with multiple sources
- Consistency checks across variables
- Peer review of methodology
- Version control via Git
